# 家庭版任务规划助手小程序 (FamilyTaskAssistant)

## 一、产品定位
**1. 核心定位**
家庭场景轻量化任务规划与协作工具，以“家庭为单位”，解决家庭成员间任务分配、进度同步、责任落地、习惯养成的核心痛点。

**2. 目标用户**
- **核心用户**：家庭管理者（父母、家庭主理人）
- **次要用户**：全家庭成员（老人、孩子、配偶）
- **细分场景**：日常家务分配、孩子学习/生活任务管理、家庭采购计划、纪念日/重要事项提醒、家庭习惯养成。

**3. 核心价值**
- **协作高效**：任务实时同步，责任到人。
- **管理简单**：零学习成本，老少皆宜。
- **习惯养成**：打卡、积分激励。
- **数据沉淀**：家庭生活轨迹复盘。

## 二、产品核心功能架构

### （一）基础层：家庭账号体系
- **家庭创建与加入**：邀请码/链接邀请，微信一键加入。
- **角色管理**：管理员（创建者）与普通成员。
- **成员信息**：支持儿童模式（简化）、老人模式（大字/精简）。

### （二）核心层：任务管理模块
- **任务创建**：支持文字+图片，设置类型、执行人、时间（单次/重复）、优先级、积分。
- **任务列表**：
  - 「我的任务」：待办/进行中/已完成/逾期。
  - 「家庭任务」：全家任务总览。
  - 「逾期任务」：标红突出。
- **执行与反馈**：一键完成（上传凭证），申请延期/取消，管理员审核。
- **实时提醒**：微信服务通知推送。

### （三）激励层：积分与奖励体系
- **积分规则**：按优先级/难度积分，额外奖励（提前/连续），惩罚（逾期）。
- **奖励中心**：自定义奖励兑换，积分排行榜。

### （四）辅助层：数据统计与特色功能
- **数据看板**：完成率、参与度、趋势分析。
- **家庭采购清单**：关联任务，共享清单。
- **纪念日**：重要事项提醒。
- **习惯打卡**：预设习惯与自定义习惯。

### （五）设置层
- 家庭信息修改、提醒设置、模式切换、数据导出、隐私设置。

## 三、产品迭代规划

### V1.0 基础版（核心功能，快速上线）
- 核心：家庭创建/加入、成员管理、单次/重复任务创建、我的任务列表、任务完成/提醒、基础微信推送。
- 目标：解决核心痛点，满足基础使用。

### V2.0 激励版（提升活跃度）
- 新增：积分体系、排行榜、奖励兑换、任务审核。
- 优化：儿童/老人模式、筛选、界面简化。

### V3.0 生态版（拓展场景）
- 新增：采购清单、纪念日、习惯打卡、数据看板、导出。

## 四、技术栈规划 (建议)
- **前端**：微信小程序原生开发 (WXML, WXSS, JS/TS) 或 Taro/Uni-app。
- **后端**：微信云开发 (Cloud Functions, Cloud Database, Cloud Storage) - 免服务器运维，适合家庭轻量级应用。
